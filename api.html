API Code Asm

Основные функции

asm_execute()

Выполняет ассемблерный код.

int asm_execute(const char* assembly_code);
  • assembly_code — строка с ассемблерными инструкциями.
  • Возвращает: код выполнения (0 — успех).

asm_register_read()

Считывает значение из регистра.

uint64_t asm_register_read(const char* register_name);
