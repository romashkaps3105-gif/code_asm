Установка Code Asm

Шаг 1: Скачайте библиотеку

Получите последнюю версию с главной страницы.

Шаг 2: Распакуйте архив

Извлеките файлы в папку вашего проекта.

Шаг 3: Подключите заголовочный файл

#include "code_asm.h"

Шаг 4: Настройте компилятор

Убедитесь, что ваш компилятор поддерживает inline assembly.
